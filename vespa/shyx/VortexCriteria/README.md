# VortexCriteria (涡流识别判据)

作为一名在代码之海里冲浪多年的全栈老司机，很高兴为您解析这个流体力学可视化领域的“神器”——`vtkVortexCriteriaFilter`。不要被它长长的名字吓到，它的核心使命非常明确：**从流体的速度场中揪出那些调皮的涡流（Vortices）**。

下面，让我们用一杯咖啡的时间，深入了解这个过滤器的内功心法。

---

## 1. 目的与核心功能算法解析 🧠

### 目的与功能
在计算流体力学（CFD）中，光看速度场往往是一团乱麻。`vtkVortexCriteriaFilter` 的终极目标是通过提取各种高级标量和张量场（如 Q-criterion、$\lambda_2$ 等），帮助我们精准定位和可视化流场中的三维涡旋结构。简单来说，它就是一台“涡流显微镜”，能够透视流体中的龙卷风！

### 核心算法原理解析
算法的灵魂在于对**速度梯度张量（Velocity Gradient Tensor, $J = \nabla u$）**的解剖：
1. **获取梯度**：通过底层的 `vtkGradientFilter`，对输入的3D速度场求空间导数，得到 $3 \times 3$ 的雅可比矩阵 $J$。
2. **张量分解**：将 $J$ 优雅地撕开，分为两部分：
   - **对称部分 (Strain Rate, $S$)**：代表流体的形变、拉伸和剪切。$S = \frac{1}{2}(J + J^T)$。
   - **反对称部分 (Rotation Tensor, $O$)**：代表流体的刚体旋转。$O = \frac{1}{2}(J - J^T)$。
3. **推导各式花招（判据）**：基于 $S$ 和 $O$，代码上演了各种数学魔法。比如，当旋转 ($O$) 战胜了形变 ($S$) 时，我们就认为此处有涡旋存在。通过求解特征值、计算矩阵范数以及向量点乘，算法一口气输出了从经典（Q准则）到前沿（Liutex）的多种物理量。

## 2. 核心参数一览表 🎛️

在这里，你可以像调音师一样拨动开关，决定要捕捉哪些流体特性。（以下布尔开关，0为关，1为开）

| 参数名称 | 默认值 | 效果与物理含义（说人话版本） |
| :--- | :--- | :--- |
| `VelocityArrayName` | `nullptr` | **速度场名称**：你要分析的3D速度场叫啥？不填的话，它会自动去找第一个三分量的数组凑合用。 |
| `Epsilon` | `1e-10` | **安全小常数 ($\epsilon$)**：防止除法时分母为零引发宇宙毁灭的小数字。 |
| `ComputeVelocity` | `0` (Off) | **输出速度**：是否把原始速度场顺手输出，方便下游做流线图。 |
| `ComputeVorticity` | `1` (On) | **计算涡量 (Vorticity)**：流体微团的自转角速度。最经典，但也容易把剪切层误认为涡。 |
| `ComputeGradient` | `0` (Off) | **速度梯度矩阵**：输出完整的 $3 \times 3$ $J$ 矩阵。 |
| `ComputeStrainRate` | `0` (Off) | **应变率张量 ($S$)**：看流体哪里被拉扯得最狠。 |
| `ComputeRotationTensor` | `0` (Off) | **旋转张量 ($O$)**：单纯的旋转部分矩阵。 |
| `ComputeQCriterion` | `1` (On) | **计算 Q 准则 (Q-criterion)**：大名鼎鼎的 $Q = \frac{1}{2}(\|O\|_F^2 - \|S\|_F^2)$。$Q>0$ 说明旋转占优，是画等值面最常用的“爆款”指标。 |
| `ComputeLambda2` | `0` (Off) | **计算 $\lambda_2$ 准则**：寻找 $S^2 + O^2$ 的第二大特征值。$\lambda_2 < 0$ 的地方通常藏着深邃的涡核。 |
| `ComputeSwirlingStrength` | `0` (Off) | **旋转强度 ($\lambda_{ci}$)**：速度梯度的复特征值的虚部。能很好地剔除纯剪切的干扰，只留下纯正的旋转。 |
| `ComputeLiutex` | `0` (Off) | **Liutex (又名 Rortex)**：比较新的前沿判据，输出一个三分量向量。它号称彻底剥离了剪切污染，只代表流体中真正的刚体旋转，流体力学界的“理中客”。 |
| `ComputeOmegaMethod` | `0` (Off) | **Omega 方法 ($\Omega$)**：$\Omega = \frac{\|O\|_F^2}{\|S\|_F^2 + \|O\|_F^2 + \epsilon}$。它是一个归一化的指标（0到1之间），通常取 $\Omega = 0.52$ 就能完美提取涡流，不用再盲猜阈值了！ |
| `ComputeHelicity` | `0` (Off) | **螺旋度 (Helicity)**：速度与涡量的点乘。附赠归一化螺旋度（-1到1）。用来衡量涡管像麻花一样“扭曲”和“打结”的程度。 |
