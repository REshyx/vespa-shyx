# vtkCGALSurfaceToVolumeMesh (曲面到体网格生成器)

欢迎来到 `vtkCGALSurfaceToVolumeMesh` 的世界！这个模块的主要工作是：给空心的表面网格“注入口味纯正的四面体夹心”。

## 1. 目的、功能与算法详述

**🎯 目的与功能：**
在计算机图形学和有限元分析中，我们经常能拿到一个漂亮但“内心空虚”的三维表面模型（比如一个水壶的外壳）。本模块的作用就是将这样一个封闭的、完全由三角形组成的表面网格（`vtkPolyData`），转化为一个内部填满四面体单元的实体体积网格（`vtkUnstructuredGrid`）。这就好比把一个空心的巧克力球变成了实心的硬核巧克力球。

**🧠 核心算法（Delaunay Refinement）：**
本模块在底层抱紧了 **CGAL (Computational Geometry Algorithms Library)** 的大腿，特别是其强大的 `Mesh_3` 核心包。
- **输入要求：** 输入的表面必须是“滴水不漏”的（Watertight/Closed）、流形的，并且没有自相交。
- **算法流程：** 首先，算法会将 VTK 的数据结构转换为 CGAL 所需的 `Polyhedral_mesh_domain_with_features_3` 域。随后，它采用 **受限三维Delaunay细分（Delaunay Refinement）** 算法。该算法通过不断在网格的内部和边界上“见缝插针”地插入新的顶点，使得最终生成的四面体网格不仅完美贴合原始边界，而且在尺寸和形状上严格满足我们设定的各项苛刻标准。

---

## 2. 参数列表及其效果和含义

要烤出完美的“四面体夹心”，你需要掌握以下“火候”控制参数：

| 参数名 | 默认值 | 含义与效果描述 |
| :--- | :---: | :--- |
| **`FacetAngle`** | 25.0 | **表面三角面片最小角度（度）：** 控制表面网格的形状质量。下限约束，值越小，算法在逼近复杂表面时生成的网格就越精细。 |
| **`FacetSize`** | 0.15 | **表面面片最大尺寸：** 控制表面 Delaunay 球的半径上限。值越小，表面网格被切分得越碎、越密集。 |
| **`FacetDistance`** | 0.008 | **表面逼近误差限制：** 表面面片外心到其 Delaunay 球心的最大距离。这决定了生成的网格与原始曲面有多“贴合”，值越小，逼近越精确。 |
| **`CellRadiusEdgeRatio`** | 3.0 | **体网格形状质量（长宽比限制）：** 四面体外接球半径与其最短边长的比值上限。用于消灭极其狭长、畸形的四面体。**注意：** 必须 > 2.01，否则算法可能会陷入死循环。 |
| **`CellSize`** | 0.0 | **体网格最大尺寸：** 控制内部四面体外接球的半径上限。设为 `0.0` 时表示不作强制限制（由表面网格的密度自然过渡）；若设为大于0的数值，则可强制约束体网格的最大体积，用于生成更粗或更细的内部网格。 |
| **`DetectFeatures`** | ON | **特征检测开关：** 开启后，算法会自动火眼金睛地识别出输入模型上的“锋利边缘”和“尖角”。如果你输入的是一个圆滑的球体，可以将其关闭以节省计算时间。 |
| **`FeatureAngle`** | 60.0 | **特征检测角度（度）：** 当相邻两个面的法线夹角大于等于此值时，它们交界的那条边就会被标记为“特征边”（需要被重点保护）。*仅在 DetectFeatures 为 ON 时有效。* |
| **`EdgeSize`** | 0.0 | **特征边分段最大长度：** 控制特征边上的采样密度。值越小，尖锐边缘上的网格节点就越密集。设为 `0.0` 表示无显式限制。*仅在 DetectFeatures 为 ON 时有效。* |
