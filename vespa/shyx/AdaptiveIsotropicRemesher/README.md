# vtkSHYXAdaptiveIsotropicRemesher (自适应各向同性重网格化)

## 🎯 目的与功能算法详细解释

**目的与功能：**
这段代码实现了一个基于曲率感知的**自适应各向同性重网格化 (Adaptive Isotropic Remeshing)** 算法。简单来说，就是给你的3D模型“换一身更合身的衣服”。如果模型表面是平坦的大平原，它会用较大的均匀三角形来覆盖以节省开销；而遇到山谷、悬崖等曲率较高（细节较多）的区域，它则会自动采用更细小密集的三角形来精准还原细节。同时，它还能像尽职的保镖一样，保护住模型原有的尖锐特征边不被破坏。

**算法流程：**
本算法底层重拳出击，直接调用了强大的 **CGAL (6.0+)** 几何算法库：
1. **特征检测 (Feature Detection)：** 首先扫描整个网格，根据设定的角度阈值（`ProtectAngle`），找出所有“棱角分明”的特征边，并在后续操作中给它们打上“免死金牌”（保护约束），防止它们在网格重构时被无情地平滑掉。
2. **构建自适应尺寸场 (Adaptive Sizing Field)：** 结合给定的曲率误差容限（`AdaptiveTolerance`）以及我们设定的边长红线（最大/最小边长），为模型表面量身定制一套“尺寸分布图”。在这张图上，曲率越大的地方，被分配到的三角形尺寸就越小。
3. **各向同性重网格化 (Isotropic Remeshing)：** 根据上一步算出的自适应尺寸场，对网格进行多次迭代的各向同性重网格化操作（包含顶点插入、边折叠、边翻转以及3步平滑松弛），最终输出一个高质量、大小过渡自然的网格模型。

## 🎛️ 参数列表及其效果和含义

要想调教好这个算法，你得熟悉以下几个“遥控器按钮”：

| 参数名称 | 类型 | 默认值 | 效果和含义 |
| :--- | :--- | :--- | :--- |
| **`MinEdgeLength`** | `double` | `-1.0` | **最小边长限制**。重网格化后三角形边的最短下限。如果设置为 `<= 0`，系统会很贴心地自动计算为输入模型包围盒对角线长度的 **0.5%**。 |
| **`MaxEdgeLength`** | `double` | `-1.0` | **最大边长限制**。重网格化后三角形边的最长上限。如果设置为 `<= 0`，则自动计算为模型包围盒对角线长度的 **5%**。在平坦区域，网格边长会尽量逼近这个值。 |
| **`AdaptiveTolerance`** | `double` | `0.001` | **自适应容差**。这是控制网格精细度的核心“魔法值”！它控制了离散曲率的误差容限。该值必须严格大于 0。**值越小，系统对曲率变化越敏感**，会在规定的最大最小边长范围内生成更细密的网格来拟合曲率。 |
| **`ProtectAngle`** | `double` | `45.0` | **特征边保护角度阈值（度）**。当两个相邻面的法线夹角大于此阈值时，它们交接的边将被判定为特征边（例如物体的直角边缘）。这些边在重网格化过程中会受到严密保护，确保模型不丢失其硬朗的几何特征。 |
| **`NumberOfIterations`** | `int` | `3` | **迭代次数**。CGAL 执行各向同性重网格化的循环次数。次数越多，网格质量通常越好，三角形越接近完美的等边三角形，但计算时间也会相应增加。该值必须 `>= 1`。 |

*(注：系统还提供了一个名为 `RefreshSuggestedEdgeLengthsFromBounds()` 的方法，专为 ParaView UI 界面设计。点击按钮即可一键获取基于当前模型尺寸推荐的默认最大/最小边长。)*
