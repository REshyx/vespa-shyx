# vtkSHYXClebschMapFilter

## 目的与功能算法详细解释

### 目的与功能
`vtkSHYXClebschMapFilter` 是一款基于 SIGGRAPH 2017 论文《Inside Fluids: Clebsch Maps for Visualization》所实现的流体可视化滤镜。它的主要目的是将流体场中的速度场（$\eta_0$）映射到四维空间的三维球面（$S^3$）上的波函数（$\psi$）。通过这种映射，算法能够生成无奇点的流体表面和纹理，极为直观、优雅地可视化流体中的涡管（Vortex Tubes）与复杂拓扑结构。该滤镜接收一个带有向量数据（如“Velocity”）的四面体网格，并输出一个包含四个分量的波函数数组 `Clebsch_Psi_S3`。

### 算法流程
1. **初始化 (Initialization)**：利用设定的随机种子，为网格中的每一个顶点随机生成复数波函数 $\psi \in \mathbb{C}^2$（映射在 $S^3$ 球面上），并进行归一化。
2. **离散外微分预计算 (DEC Precomputation)**：遍历输入的四面体网格，计算各个顶点的质量矩阵（Mass Matrix，$M_V$）、边权重（$w_{ij}$）以及速度场沿边的积分（$\eta_0$）。在此过程中使用了几何雅可比矩阵（Jacobians）来构建离散拉普拉斯算子。
3. **参数自适应 (Auto-tuning $\hbar$)**：如果开启了 `AutoHbar`，算法会根据网格的平均边长和最大速度自动估算普朗克常数 $\hbar$ 的最佳值，确保可视化结果的频率适中。
4. **迭代求解器 (Iterative Solvers)**：
   - 算法包含双层嵌套循环。外层循环由参数 `epsilon` 控制（每次迭代衰减为原来的 $1/10$），用于逐步逼近最优解。
   - 内层循环根据当前的波函数状态，构建一个巨大的复数稀疏埃尔米特（Hermitian）线性系统 $Ax = b$。
   - 如果启用了 `UseMKL`，算法将优先使用 Intel MKL 的 Pardiso 求解器（LLT, LDLT, LU）进行直接求解，以加速大规模矩阵运算；否则，将使用共轭梯度法（Conjugate Gradient, CG）进行迭代求解。
   - 每次求解结束后，波函数会被更新并再次投影归一化到 $S^3$ 空间。
5. **输出结果**：将优化后的 $\psi$ 转换为 `Clebsch_Psi_S3` 数组，附加到输出数据的点属性中，供后续的渲染管线（如提取等值面、着色）使用。

## 参数列表及其效果和含义

| 参数名称 | 类型 | 默认值 | 效果与含义 |
| :--- | :---: | :---: | :--- |
| **Hbar** ($\hbar$) | `double` | 0.1 | 缩放参数，控制 Clebsch 映射的空间频率。较小的 $\hbar$ 会生成更密集的干涉条纹（高频），较大的 $\hbar$ 则生成较稀疏的条纹。 |
| **AutoHbar** | `bool` | true | 自动估算 $\hbar$。开启后，滤镜将根据网格平均边长和最大速度场自动覆盖当前的 `Hbar` 值，建议在不了解合适范围时保持开启。 |
| **DeltaT** ($\Delta t$) | `double` | 1.0 | 迭代优化过程中的时间步长。该值影响非线性优化的收敛速度与稳定性。 |
| **MaxIterations** | `int` | 20 | 内层循环的最大迭代次数（线性系统求解次数）。数值越大，单次外层循环逼近最优解的程度越高，但会增加计算耗时。 |
| **OuterLoops** | `int` | 5 | 外层松弛循环次数。控制整体算法的优化深度，`epsilon` 参数在此循环中逐步衰减，有助于避免陷入局部最优。 |
| **RandomSeed** | `int` | 42 | 初始化波函数时使用的随机种子。固定该值可以确保多次运行算法生成一致的波函数场（保证可重复性）。 |
| **VelocityArrayName** | `string` | "Velocity" | 输入数据集中用于表示速度场的点向量数组名称。 |
| **UseMKL** | `bool` | false | 性能优化开关。若设为 `true` 并在编译时链接了 Intel MKL 库，将调用 Pardiso 直接求解器处理稀疏矩阵，显著提升求解速度。 |
